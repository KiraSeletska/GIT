 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

- На GitHub открыть репозиторий, который нужно склонировать.
- Скопировать SSH ключь (зеленая кнопка <Code>)
- В терминале на комьютере открыть необходимую папку, кдуа необходимо клонировать репозиторий из github.
- Ввести команду: git clone и вставить скопированный SSH-ключ.

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

- Создаем репозиторий на github, копируем ключь. 
- git init - в консоли (находясь в нужной директории) инициализируем локальный репозиторий.
- git status - выводим файлы, которые еще не отслежиываются.
- git add - добавляем нужные файлы их имена и расширения, или все командой - git add .
- git commit -m ‚название’ - называем исходя из действий
- git branch -M main - переключаемся на главную ветку
- git remote add origin <SSH-ключ> - привязываем репозиторий 
- git push -u origin main - отправляем файлы 

3. Как получить на локальный компьютер изменения с github.com.

- выполняем команду git pull

4. Какую информацию нам дают команда :
git status
git remote -v 

 - git status - показывает наличие измененных файлов в локальном репозитории 
 - git remote -v - показывает подключенные удаленные репозитории и их протоколы
