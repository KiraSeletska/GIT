Описание занятия: 
1.Как отказаться от изменений в файле (файлах) находящийся в 
рабочем каталоге (work directory)?
 -git checkout (имя файла)
2.Как вернуть файл из индекса в рабочий каталог?
 -git restore —stage (имя файла)

3.Как отменить ошибочный commit ?
 -git revert HEAD 

4.Как создать новую ветку и переключится в нее?
 -git checkout -b (имя ветки)

5.В чем отличение между fast-forward merge  and non fast-forfard merge?
 - Fast-forward merge происходит при отсутствии расхождений (дополнительных коммитов)
 с текущей веткой. 
 - Non Fast-forward merge - ошибка, которая указывает, на невозможность сохрранить 
изминения (коммит утерян или ведется с паралельная работа с веткой)

6.Можно ли удалить ветку? Как? 
Да, использовав комманду:
git branch -d (название ветки)
